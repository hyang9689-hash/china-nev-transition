# Past the Tipping Point, Not Yet Transformed

**China's new-energy vehicle sales, fleet, and charging transition**

This repository is a reproducible research project about a concrete change in
China's automobile market: new-energy vehicles are becoming dominant in new
sales much faster than they are replacing the existing vehicle fleet.

## Research question

How quickly is China's dominance of new-energy vehicles in new-vehicle sales
translating into the on-road fleet, and has charging infrastructure kept pace?

## Status

Day 1 foundation. Data, analysis, notebook, report, tests, and a publication
workflow will be added as separate, traceable commits.
